# Deploy to GitHub Pages
# 1. Install: npm install -D gh-pages
# 2. Run: npm run deploy:github
# 3. Make sure your repo remote is set correctly
# 4. Enable GitHub Pages on the `gh-pages` branch in repo settings

# Deploy to Firebase Hosting
# 1. Install: npm install -g firebase-tools
# 2. Login: firebase login
# 3. Init: firebase init hosting (replace .firebaserc project id)
# 4. Run: npm run deploy:firebase
